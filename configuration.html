Steps to Configure FTP

  1. Create two distinct networks: 192.168.1.0 and 192.168.2.0 (Class C).
  2. Connect essential networking devices such as routers, switches, and PCs.
  3. Assign static IP addresses, subnet masks, and default gateways to all devices.
  4. Enable Routing Information Protocol (RIP) on all routers for dynamic inter-network communication.
  5. Set up the FTP server with user credentials and appropriate file permissions (read, write, delete, rename, list).

Routing Configuration (RIP)

Configure RIP on the routers to advertise their connected networks:

  • Router1 Networks: 192.168.1.0, 10.0.0.0, 11.0.0.0
  • Router2 Networks: 100.0.0.0, 12.0.0.0, 11.0.0.0
  • Router3 Networks: 192.168.2.0, 12.0.0.0, 11.0.0.0

FTP Commands

ftp [IP_ADDRESS] – Connect to the FTP server.

dir – List available files on the server directory.

put [FILENAME] – Upload a local file to the server.

get [FILENAME] – Download a file from the server to your local machine.
