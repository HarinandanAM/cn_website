What is FTP?

The File Transfer Protocol (FTP) is a standard network protocol used to transfer files between a client and a server on a computer network. It enables users to upload and download files over TCP/IP-based networks such as the Internet or an intranet. FTP uses separate control and data connections between the client and the server, allowing users to authenticate, manage file directories, and execute transfer commands.

In educational and lab environments like Cisco Packet Tracer, FTP simulation is crucial for understanding client-server file interactions, permission settings, and network routing.

About the Project

In this project, a basic FTP setup is configured across two networks (192.168.1.0 and 192.168.2.0). Using Cisco Packet Tracer, we demonstrate how users can upload and download files across these networks through a centralized FTP server. Routing Information Protocol (RIP) is used for dynamic routing between the routers to ensure devices can communicate seamlessly.
